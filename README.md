# Student_Performance_Analyzer
A Python-based data analysis project that evaluates student academic performance using Pandas and visualizes insights using Matplotlib. The results are displayed through a clean HTML dashboard interface.
#Project Overview
The Student Performance Analyzer processes a dataset of student marks and attendance details to generate meaningful academic insights.
This project demonstrates:
Data processing using Python
Data analysis using Pandas
Visualization using Matplotlib
Basic frontend integration using HTML
Logical problem-solving and debugging
#Features
Automatic calculation of:
Total Marks
Average Marks
Grade Assignment
Identifies class topper
Calculates overall class average
Pass/Fail classification
Attendance percentage tracking
Bar chart and Pie chart visualizations
Simple HTML dashboard for displaying results
